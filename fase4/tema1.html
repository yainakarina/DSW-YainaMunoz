    Las CSS (coding style sheets) son hojas de estilo en cascada, que permiten aplicar una o más reglas para dar una apariencia a cada elemento de una página web. Se pueden aplicar diferentes hojas de estilo para una página web, dependiendo de los dispositivos en los que se vea la misma.

    Cada regla de las hojas de estilo se compone de dos partes:

  • Selector: indica a cuáles elementos se les va a aplicar la regla. Se pueden escribir varios selectores para una misma regla, separados por coma (,).
  • Lista de declaraciones: son los estilos que se van a aplicar. Propiedad:valor, separados por punto y coma (;).

Principales listas de declaraciones


    Las listas de declaraciones establecen las propiedades visuales de los elementos seleccionados. Las declaraciones CSS admiten las unidades de porcentaje (tamaño en un tanto por ciento), em (tamaño de acuerdo al tamaño de la fuente actual o heredada), px (tamaño en pixeles), cm (centímetros), mm (milímetros), in (pulgadas), pt (puntos):

  • Propiedades de texto: color, tamaño de fuente, letra cursiva, negritas, subrayados y tachados, etc.
  • Propiedades de fondo
  • Dimensiones y bordes: ancho y alto de un elemento, margen externo e interno del elemento con respecto a otro.
  • Propiedades de lista: tipo de marcador para una lista, si el marcador se incluye dentro o fuera del texto del elemento, añadir una imagen como marcador
